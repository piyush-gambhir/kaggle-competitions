# Kaggle Competitions

Notebooks and submissions for Kaggle competitions.

## Competitions

- **Titanic — Machine Learning from Disaster** — binary classification on the classic Titanic dataset
- **Digit Recognizer** — handwritten digit classification (MNIST)

## Stack

Python, scikit-learn, TensorFlow / Keras, pandas, NumPy. Each competition lives in its own directory with the notebook, the dataset (or a download script), and submission CSVs.

## Running

```bash
cd "Titanic - Machine Learning from Disaster"
jupyter notebook
```
